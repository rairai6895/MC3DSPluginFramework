# MC3DSPluginFramework

「Minecraft New Nintendo 3DS Edition」向けのプラグイン開発用フレームワークです。  
このライブラリには、リバースエンジニアリングしたゲーム内部のクラスや関数情報などがまとめられており、Modやプラグイン開発に利用できます。

---

## 内容
- ゲーム内クラス・関数の逆コンパイル情報と互換性のある再実装
- イベントをトリガーするためのフック
- バグ修正用のフック
- コーディングを楽にするユーティリティ・ヘルパー
- 拡張機能

---

## 注意
- アップデートにより関数名・引数などが変更されることがあります
- エミュレーターで開発しているため、一部機能が実機で動作しない場合があります
- `LightLock` (Mutex) を使用する関数は、ゲームスレッド停止中に使用するとフリーズする可能性があります

---


## 使い方
`MC3DSPluginFramework.hpp` をインクルードして使用してください。
